RAG Pipelines
========================================


Genetic
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The :class:`pyterrier_rag.Genetic` pipeline performs RAG by using a genetic algorithm to iteratively
construct a strong answer.

The pipeline expects a result frame as input, and returns an answer frame.
